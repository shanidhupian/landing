# Images Directory

This directory contains all images for the Shani Dhupian website.

## Directory Structure

```
images/
├── hero/
│   ├── hero-bg.webp      # Hero section background image (WebP format)
│   └── hero-main.webp    # Main hero image (optional, WebP format)
├── products/
│   ├── cotton-fabric.webp    # Cotton fabric product image (WebP format)
│   ├── lycra-fabric.webp     # Lycra fabric product image (WebP format)
│   └── bakery-box.webp       # Bakery box product image (WebP format)
└── about/
    ├── about-us.webp         # About section main image (WebP format)
    └── business-info.webp     # Business information image (WebP format)
```

**Note:** Images from IndiaMART are in WebP format. Use `.webp` extension for all images.

## Image Requirements

### Recommended Specifications:

**Hero Images:**
- `hero-bg.webp`: 1920x1080px (16:9 aspect ratio)
- `hero-main.webp`: 800x600px or larger
- Format: WebP (from IndiaMART)

**Product Images:**
- Size: 800x600px or larger
- Format: WebP (from IndiaMART)
- Aspect Ratio: 4:3 or 16:9

**About Images:**
- Size: 1200x800px or larger
- Format: WebP (from IndiaMART)

## Adding Images

1. Place your images in the appropriate subdirectories
2. Use the exact filenames listed above
3. Ensure images are optimized for web (compressed but good quality)
4. The website will automatically display placeholder graphics if images are missing

## Image Optimization Tips

- **WebP format**: IndiaMART images are already in WebP format, which is optimized
- WebP provides excellent compression while maintaining quality
- No additional optimization needed for WebP images from IndiaMART
- If converting from other formats, use tools like Squoosh or ImageMagick
